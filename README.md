# 🌐 Decentralized DNS on Stellar (Soroban)

## 📌 Project Description

This project implements a **Decentralized Domain Name System (DNS)** using **Soroban smart contracts on the Stellar blockchain**. It allows users to map human-readable domain names (like `ankit.stellar`) to blockchain addresses in a secure and censorship-resistant manner.

## ⚙️ What It Does

Traditional DNS systems are centralized and controlled by authorities. This project replaces that with a **blockchain-based DNS**, where:

* Users can **register domain names**
* Domains are mapped to **wallet addresses**
* Data is stored **on-chain**
* No central authority controls domain ownership

---

## 🚀 Features

* 🔐 **Decentralized Ownership**
  Users fully control their domain registrations.

* 🧾 **Domain Registration**
  Register a unique domain name mapped to an address.

* 🔎 **Domain Resolution**
  Convert domain names into wallet addresses.

* 🔄 **Update Records**
  Modify existing domain mappings.

* ⚡ **On-chain Storage**
  All records are securely stored on Stellar via Soroban.

## 📦 Smart Contract Functions

| Function                      | Description            |
| ----------------------------- | ---------------------- |
| `register(domain, address)`   | Registers a new domain |
| `resolve(domain)`             | Returns mapped address |
| `update(domain, new_address)` | Updates domain mapping |
